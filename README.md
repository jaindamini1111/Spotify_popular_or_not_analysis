# Spotify_popular_or_not_analysis



Using the top Spotify songs dataset (2010-2019) from Kaggle predict if the song will be able to make it to hit list (top 30s) or not.


Libraries Used: Pandas, Numpy, Sklearn, Seaborn, Matplotlib, Scipy,stats, wordcloud.



Data Source: https://www.kaggle.com/leonardopena/top-spotify-songs-from-20102019-by-year/kernels
